## Compile instructions
Ubuntu: Install prerequisites with:

```
sudo apt-get install build-essential cmake libqt5-dev qt5-qmake libqt5svg5-dev
```

Windows: Just use CMake as usual. Required: Visual C++ 2008 or newer.
